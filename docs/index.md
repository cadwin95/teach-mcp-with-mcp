## 📝 소개

**MCP(Model Context Protocol)**는 다양한 AI 모델, 외부 툴, DB 등을 일관된 방식으로 연결해주는 오픈 프로토콜입니다.  
OpenAI function-calling 등과 유사하지만, 멀티 모델/멀티 툴/멀티 에이전트 통합이라는 강력한 확장성이 특징입니다.
